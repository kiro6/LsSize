# LsSize
Tool to list the sizes of all files and folders in a directory


## Example
```bash
┌──(test㉿test)-[~/Desktop/tests]
└─$ lsSize
File: html.html (72K)
File: index.php (4.0K)
File: js.js (4.0K)
Directory: New Folder (4.0K)
File: py.py (4.0K)
File: sh.sh (4.0K)
File: test2.js (0)
File: Text File.txt (4.0K)
File: vars.php (4.0K)
Total: /home/test/Desktop/tests (120K) 
```
